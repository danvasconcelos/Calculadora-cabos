# Aplicativo de Dimensionamento de Cabos #

## Visão Geral ##

O aplicativo é uma ferramenta digital de suporte à decisão, projetada para eliminar o erro humano no cálculo de capacidade de carga de cabos de aço. Ele permite que os técnicos de manutenção preventiva e corretiva determinem, de forma instantânea e precisa, o diâmetro ideal do cabo para cada operação de içamento de máquinas e equipamentos.

### Funcionalidades Técnicas ###

O algoritmo processa múltiplas variáveis que, se calculadas manualmente sob pressão, poderiam gerar falhas:

•	Cálculo de Tensão por Ângulo: Ajusta automaticamente a capacidade de carga com base no ângulo das pernas (0° a 60°).

•	Configuração de Pernas: Suporta simulações para 1, 2 ou 4 pernas de içamento.

•	Modos de Engate: Diferencia içamentos Diretos, em Cesto (Basket) ou Enforcado (Choker) — este último aplicando automaticamente a redução de 20% na capacidade conforme exigido por norma.

•	Especificação de Alma: Permite escolher entre cabos com alma de fibra (AF) ou aço (AA).
